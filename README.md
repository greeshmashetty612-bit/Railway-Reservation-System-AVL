# Railway Reservation System

A console-based Railway Reservation System developed in C using AVL Trees for efficient passenger, coach, and seat management.

## Features

- Ticket Booking
- Ticket Cancellation
- Waiting List Management
- Passenger Details Management
- Display Available Seats
- Display Passengers
- Passenger Search
- Range Search
- File-based Data Persistence
- Input Validation

## Data Structures & Concepts Implemented

- AVL Trees
- Binary Search Trees
- Tree Rotations
- Recursive Tree Traversals
- Dynamic Memory Allocation
- File Handling
- Searching & Sorting
